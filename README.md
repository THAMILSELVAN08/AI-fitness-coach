# 🏋️ AI Fitness Coach

AI Fitness Coach is a simple web application that generates personalized fitness plans using Google's Gemini AI.

---

## 🚀 Features

- 👤 User profile creation (name, age, weight, height, workout duration, target areas)
- 🤖 AI-powered fitness plan generation using Gemini API
- 🎨 Clean and responsive web interface
- ⚠️ Error handling for API failures
- 🔐 Session-based user management

---

## 📂 Project Structure

- **main.py** – Flask web application (includes HTML & CSS template)
- **gemini_api.py** – Gemini API integration module
- **README.md** – Project documentation

---

## 🛠 Prerequisites

Before running the project, make sure you have:

- Python 3.7 or higher
- A Google Gemini API Key
